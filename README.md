# REMIT Agent Audit Standard

> **Status:** Draft — canonical specification text will be published here as the standard matures.

Requirements for producing tamper-evident, auditable records of agent actions and decisions.

Development takes place in the open via the [agent-governance working group](http://remit.test/working-groups/agent-governance) and associated GitHub discussions.
